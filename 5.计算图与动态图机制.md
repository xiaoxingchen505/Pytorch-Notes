

# 计算图与动态图机制 Computational Graph

计算图是用来描述运算的有向无环图


计算图有两个主要元素：结点 Node，边 Edge

结点表示数据，如向量，矩阵，张量，边表示运算，如加减乘除卷积等。

用计算图表示: y = (x+w) *(w+1)

a = x+w, b = w+1

y = a * b

计算图与梯度求导:

dy/dw = dy/da * da/dw + dy/db * db/dw

      = b * 1 + a * 1
      = b + a
      = (w+1) + (x+w)
      = 2*w + x + 1
      = 2*1 + 2 + 1  = 5
